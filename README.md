# 🏡 Florida Real Estate Sales Analysis Dashboard (Excel)

## 📌 Overview

This project focuses on analyzing a real-world Florida real estate dataset using Microsoft Excel. The goal was to transform raw, messy data into meaningful insights through data cleaning, feature engineering, and interactive dashboard creation.

---

## 📂 Dataset

* Source: Kaggle (Florida Real Estate Sold Dataset)
* Data includes:

  * Property Type
  * List Price & Sold Price
  * Square Footage
  * Bedrooms & Bathrooms
  * Garage & Stories
  * Year Built & ZIP Code

---

## 🧹 Data Cleaning

* Removed irrelevant columns (e.g., long text descriptions)
* Handled missing values:

  * Deleted rows for critical fields (sqft, beds)
  * Filled logical values (garage = 0, stories = 1)
* Standardized numeric formats and ZIP codes
* Removed duplicates

---

## ⚙️ Feature Engineering

Created additional columns to improve analysis:

* Price Difference
* Discount Percentage
* Price per Sqft
* Property Age
* Size Category (Small, Medium, Large)

---

## 📊 Analysis (Pivot Tables)

Key analyses performed:

* Average Price by Property Type
* Price vs Bedrooms
* Price by Size Category
* Garage Impact on Price
* Listings by ZIP Code
* Top 10 Most Expensive ZIP Codes

---

## 📈 Dashboard

An interactive Excel dashboard was built using:

* Pivot Charts (Bar & Column Charts)
* Slicers for filtering:

  * Property Type
  * Bedrooms
* Clean and structured layout for easy insights

---

## 🧠 Key Insights

* Property prices increase with size and number of bedrooms
* Homes with garages tend to have higher prices
* Significant price variation exists across ZIP codes
* Certain ZIP codes represent premium real estate markets

---

## 🛠️ Tools Used

* Microsoft Excel

  * Pivot Tables
  * Charts
  * Slicers
  * Data Cleaning Functions

---

## 🚀 Project Outcome

This project demonstrates end-to-end data analysis using Excel — from raw data cleaning to building a professional, interactive dashboard.
